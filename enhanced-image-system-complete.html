🎛️ Context Menu Options

Right-click on any image or click the ⋯ button in the top-right corner to access:

📝 Edit Caption - Add or edit image captions that appear below the image
🏷️ Edit Alt Text - Modify alt text for accessibility and SEO
💧 Add Watermark - Add watermark to protect your images (placeholder for future implementation)
📱 Normal Width - Standard image size (max-width: 2xl)
📐 Wide Width - Larger image size (width: 80%, max-width: 4xl)
🖼️ Full Width - Maximum image size (full container width)
🗑️ Delete Image - Remove the image from the post

  • Wide Width: w-4/5 max-w-4xl - Great for detailed images
